Circular portrait for faculty, principal, testimonials, and alumni; shows initials on a maroon tint when no image is supplied.

```jsx
<Avatar src="/vimala.jpg" name="Vimala Jayaraj" size="lg" ring />
<Avatar name="Anita Rao" size="md" />
```

Sizes `sm | md | lg | xl` or a pixel number. `ring` adds a gold featured ring.
