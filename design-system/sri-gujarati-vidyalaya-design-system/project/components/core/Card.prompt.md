Flexible content surface for facilities, programmes, news, and faculty cards.

```jsx
<Card image="/news.jpg" eyebrow="27 Jun" title="Plus One Admission 2024-25"
      interactive footer={<a href="#">Read more →</a>}>
  Applications for the higher secondary stream are now open.
</Card>
```

Variants: `elevated | outline | soft`. Set `interactive` for hover lift + image zoom. Compose freely via `children`.
