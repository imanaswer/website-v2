Pill-shaped call-to-action button in heritage maroon — use for Apply Now, Enquire, Book a Visit and other primary actions.

```jsx
<Button variant="primary" size="lg" iconRight={<ArrowIcon/>}>Apply Now</Button>
<Button variant="secondary">Request Info</Button>
<Button variant="accent">Download Brochure</Button>
<Button variant="inverse">Book a Campus Visit</Button>
```

Variants: `primary` (maroon fill), `accent` (gold fill), `secondary` (outline), `ghost` (text only), `inverse` (cream on dark backgrounds). Sizes `sm | md | lg`. Supports `iconLeft`/`iconRight`, `fullWidth`, `disabled`, and `as="a"` with `href`.
