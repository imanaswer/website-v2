Animated count-up statistic for "School at a Glance" bands and the achievements dashboard.

```jsx
<Stat value={156} suffix="+" label="Years of Excellence" />
<Stat value={98} suffix="%" label="Board Results" tone="inverse" />
```

Counts up when scrolled into view (set `animate={false}` to disable). Use `tone="inverse"` on maroon backgrounds.
