Labelled text field with focus ring, hint and error states — for admissions enquiry, contact and search forms.

```jsx
<Input label="Parent name" required placeholder="Full name" />
<Input label="Email" type="email" hint="We'll send the prospectus here" />
<Input label="Phone" error="Enter a valid 10-digit number" />
```

Supports `icon` (leading), `hint`, `error`, and all native input props.
